# Iron Bloomery Progression Tweak

---

### This mod began as a small proof-of-concept project for friends and it serves as my first experience developing a Minecraft mod.

## Overview

Iron Bloomery Progression Tweak slows early-game iron ingot acquisition.  

* Progression up to and including **stone tools** is unchanged.
* Added a new furnace-type block called the **Bloomery**.
* The Bloomery smelts **raw iron ore into iron nuggets**, requiring additional processing to produce iron ingots.
